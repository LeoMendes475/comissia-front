## Passo a passo para rodar o projeto

Versões utilizadas
```
-> Node
v18.18.0
```
```
-> Npm
10.8.2
```

Clone o projeto
```sh
git clone https://github.com/LeoMendes475/task-manager-front.git
```
```sh
cd task-manager-front
```

Instalar dependências do projeto
```
-> npm
npm i
```
```
-> yarn
yarn dev
```

Rode o projeto
```sh
npm run dev
```
```sh
yarn dev
```

Acesse o projeto
[http://localhost:5173](http://localhost:5173)
